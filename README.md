# 🚀 My Portfolio

A modern, animated portfolio website built with Next.js, TypeScript, Tailwind CSS, and modern web technologies.

## 🌐 Live Demo

[View Portfolio](https://my-portfolio-blue-nine-61.vercel.app/)


## ✨ Features

- Modern responsive UI
- Smooth animations
- Dark/Light mode
- Interactive project showcase
- Experience timeline
- Skills section
- Contact form
- SEO optimized
- Fast performance
